Title

Verify user login with valid credentials

Description

Ensure that a registered user can successfully authenticate using their username and password. The system should issue a valid JWT token upon success.

Pre-conditions

  • User must be registered in the database.
  • System API must be online.

Steps

01
Navigate to Login Page

Open the application URL and click 'Login'.

02
Enter Credentials

Input valid username and password.

03
Submit Form

Click the 'Sign In' button.

Expected Result

User is redirected to the Dashboard. "Welcome" message is displayed.
